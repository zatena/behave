# behave
### 基于 python+behave+allure WEB UI 自动化框架，可视化测试报告。

#### 安装命令：
pip/pip3 install behave <br>

pip/pip3 install allure-behave <br>

#### 根目录执行以下命令：

1. behave -f allure_behave.formatter:AllureFormatter -o report ./features <br>
2. allure serve report
